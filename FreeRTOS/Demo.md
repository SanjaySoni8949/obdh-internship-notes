# FreeRTOS Demo Applications :-https://github.com/FreeRTOS/FreeRTOS/tree/main/FreeRTOS/Demo

FreeRTOS provides demo applications for every supported port.

These demos are useful for:

* Learning how to use FreeRTOS APIs.
* Understanding the project structure.
* Testing whether FreeRTOS is working correctly.
* Using as a starting point for new projects.

Each demo project contains:

* A pre-configured project.
* Example tasks.
* A `main.c` file that creates tasks and starts the FreeRTOS scheduler.

Most demo applications are located in:

```text
FreeRTOS/Demo/
```

Many demos also include a simple **blinky** project that creates a few tasks and demonstrates basic FreeRTOS functionality.

> **Note:** Demo applications are meant for learning and testing. They can be modified and used as the starting point for real embedded applications.
