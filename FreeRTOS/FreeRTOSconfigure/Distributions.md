# Official FreeRTOS Distribution

FreeRTOS can be downloaded either as individual libraries or as a complete distribution.

The complete distribution is recommended for beginners because it includes:

* FreeRTOS Kernel
* All supported ports
* Example projects
* Demo applications

Although the distribution contains many files, a project only uses the files required for its target processor and compiler.


Use https://github.com/FreeRTOS/FreeRTOS/releases/latest to download a zip file containing the latest
distribution.
